# Infineonの2km行列とAIインフラ時代の真のボトルネック

ドイツ系半導体メーカーの Infineon Technologies がマレーシア・マラッカで開催した大規模採用イベントが話題となった。

* オペレーターや技術者職を約400名募集
* 初任給は月額RM3,500程度（約11〜12万円相当）
* 応募者は1,000人以上集まり、行列は最大約2kmに達した
* 朝5時頃から並ぶ求職者もいた
* SNSでも大きな話題になった

Infineonの求職者の行列は半導体需要の拡大を示している。

しかし真のボトルネックは労働者数ではない。

AIインフラ時代の制約は、製造・設備・データ・顧客・資本を接続できる高次人材の不足にある。

## なぜこれが起きるのか

マレーシアでは現在、

* AI
* データセンター
* 半導体
* 先端パッケージ
* IC設計

への投資が急増している。

政府も半導体上流工程へのシフトを国家戦略として推進している。

さらに、

* Infineon
* Intel
* AMD関連企業
* AT&S

などが大型投資を進めている。

特にAT&SはAI需要を背景に、マレーシアへ最大20億ユーロ規模の追加投資を発表した。

## 雇用はある。しかし品質は足りるのか？

マレーシア政府や業界団体は現在、

* 半導体人材不足
* AI対応人材不足
* エンジニア不足

を深刻な課題として扱っている。

特に先端パッケージ、IC設計、AI対応製造の領域では、人材需要の増加速度に教育・育成が追いついていないとされている。

工場オペレーターは比較的短期間で育成可能である。

一方で、設備技術者や工程技術者になると必要な知識は急激に増加する。

さらに希少なのは、製造、設備、品質、サプライチェーン、顧客、AI需要を横断的に理解し、接続できる人材である。

## 労働力不足ではなく「接続能力不足」

このニュースから見えるのは、労働力不足ではなく「接続能力不足」である。

行列の1,000人全員が同じ価値を持つわけではない。

AIインフラ時代になるほど、価値を決めるのは人数ではなく、どれだけ複数階層を接続できるかである。

## 歴史は繰り返す

実はこれは歴史的にも珍しい現象ではない。

19世紀の産業革命でも、織機が導入されたとき、「職人は全員失業する」と言われた。

しかし実際には、

* 工場
* 鉄道
* 物流
* 機械保守

など、膨大な新職種が生まれた。

AIは労働を消しているのではない。

労働需要を別の場所へ移動させているのである。

Infineonの2km行列は、その移動の一断面に過ぎない。

AIインフラ時代の競争力を決めるのは、製造、設備、品質、顧客、資本、そしてAI需要を接続できる人材の厚みである。

行列の長さではなく、接続能力の厚みこそが次のボトルネックになる。
