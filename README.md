# AI Civilization Atlas

Observing the Structures of the AI Age

---

AI Civilization Atlas is an independent research project exploring the infrastructure, manufacturing, capital, materials, and institutions that make AI possible.

Rather than focusing only on AI models, this atlas examines the broader civilization emerging around AI.
